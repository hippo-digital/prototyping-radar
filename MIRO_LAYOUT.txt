### Step 1: Set Up Your Board

1. Create a large frame titled "Prototyping Skills Radar"
2. Divide it into 4 equal sections (quadrants)
3. Label each section with a quadrant name

### Step 2: Import Data

**Option A - CSV Import (Fastest):**
1. In Miro, click "..." (three dots) â†’ "Import from CSV"
2. Upload `miro-export.csv`
3. All sticky notes will be imported with titles, descriptions, and tags
4. Filter by tag to organize into quadrants

**Option B - JSON + Script (Advanced):**
1. Use `miro-export.json` with Miro API
2. This includes color coding for each ring level
3. Requires Miro app/API access

**Option C - Manual Entry:**
1. Use this document as reference
2. Create sticky notes for each skill
3. Apply color coding based on ring level
